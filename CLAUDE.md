## Plane (Issue 管理)

- Plane project: `codenica/DLAL` — https://plane.codenica.dev/codenica/projects/ae3b2722-419f-43c2-a165-4decabc3f8c5/issues/
- Issue 起票は Plane で `DLAL-NN` として行う (GitHub Issues / Forgejo Issues は使わない)
- commit msg は `DLAL-12 fix login error` のように Issue ID を先頭に書いて連携する
- 状況確認は Web UI、 または Claude code セッション内の plane MCP tool (`mcp__plane__list_work_items` 等)
